Saandy


Hello, I am Erik Sänd.
A Game Developer, programmer and creative born in Sweden, 1998.

I have been working in the Unity game engine and using it as a means to express my creativity since 2014 when I entered high school focusing on science and art in relation to game programming in C#.
In 2019 I entered the Blekinge Institute of Technology with a course of study focused on game programming in C++, but dropped out in 2020 at the start of the Covid pandemic.

Whilst I love programming, since dropping out of University I have been focusing less on the technical side of game development, shifting my focus to mapping in Source 2 and modelling using Maya and ZBrush.
Most recently I have delved deeper into UV mapping as well as creating both high-resolution tileable textures and materials for my models, although I still have a lot to learn on that front!
I use the models and materials I create in my Hogwarts map for Source 2, which is my latest project where I try to recreate the Hogwarts castle from the movies as accurately as I can for use in the upcoming game S&Box for roleplay and other purposes.

My Inventory

My Inventory is an unreleased inventory system for Unity I planned to release at the start of this year. The purpose of the system was to provide an easy solution to inventory management for new and even experienced game developers for use in their games. The system is quite customizable, and able to portray many different styles with custom functionality depending on the users need.

My Inventory generates a mesh based on a multitude of customizable parameters, and displays it as an Image component on the Canvas.
I plan to go back to this project in the future, as it is so close to completion, there are just a couple of things left to iron out. There is even a documentation built albeit somewhat outdated. You can find the documentation here.

Source 2 - Alyx

Ever since Valve released the tools for Half-Life: Alyx, allowing anyone to create custom levels for the game with the tools that Valve use themselves, I have been obsessed with it.
Since there are no real programming tools included, I was forced into a new path of creating maps and level art.
This has grown to be a new passion for me, and I love the process of creating environments that feel lived in, all from scratch.

I released my first playable map for Half-Life: Alyx on 24 july, 2020. You can download it here!

A continuation was briefly worked on, continuing right where Street_War ended. The map would have the player crash in an elevator, and move around in a dark environment far below the ground.
